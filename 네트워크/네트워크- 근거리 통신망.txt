네트워크- 근거리 통신망(LAN, Local Area Network)

LAN의 표준 802.X  시리즈 

1. 802.1 : LAN의 전체 구성
2. 802.2 : LLC에 관한 규약
3. 802.3 : CSMA/CD, 유선랜(충돌감지 가능)
4. 802.4 : 토큰버스
5. 802.5 : 토큰링
6. 802.11 : CSMA/CA, 무선랜(충돌감지는 불가능, 충돌회피) 
7. 802.15 : 무선 개인 통신망(WPAN) 표준 

HDLC(High-level Data Link Control)
프레임구조

플래그-주소 필드-제어 필드-데이터-FCS

플래그 : 각 프레임의 시작과 끝
주소 필드 : 프레임의 수신자를 식별
제어 필드 : 프레임의 유형(정보,감독 또는 비순서프레임)을 식별
데이터 : 실제 전송할 사용자의 데이터
FCS : 오류 검출을 위판 필드
